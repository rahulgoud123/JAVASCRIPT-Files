Loops in JS

While loop

In Javascript, a while loop is a control flow statement that allows you to repeatedly execute a block of code as long as a specified condition is true.

Do while loop

In javascript, a do-while loop is a control flow statement that repeatedly executes a block of code as long as a specified condition evaluates to true.

For loop

For loop runs through the block of code a predetermined number of times. And it works well when you know exactly how many iterations you need.

Types of for Loop:
1. for/in - loops through the properties of an object
2. for/of - loops through the values of an array, and string
