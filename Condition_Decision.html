There are 3 condition checking operators

  1. If condition
  2. if-else
  3. Switch condition

If condition

The if statement is the most basic conditional statement. It evaluates a condition and, if the condition is true, it executes a block of code.

If-else

The if-else statement allows you to execute one block of code if the condition is true and another block if the condition is false.

Ternary operator

The ternary operator is a concise way to write an if-else statement in a single line.
