Example of Functions

Functions in javascript provide organized, reusable code to perform a set of actions. Functions simplify the coding process, prevent redudant logic, and make code easier to follow.

Function parameter is a variable listed in the parentheses of a function declaration, and an argument is the actual value or expression passed to the function when it is called.
A function return statement is used to specify the value that a function should return when it is called.

Default parameters in functions allow you to specify default values for parameters in case the caller doesn't provide a value for them
