Example of Event bubbling, capturing and delegations

In the event bubbling phase, the event starts from the target element that trigerred the event and bubbles up through its ancestors in the DOM hierarchy. Most mordern browsers follow the event bubbling model by default.

In the event capturing phase, the event starts from the top-level ancestor and trickles down to the target element that trigerred the event. You can enable event capturing by setting the third parameter of addEventListener to true.

Event delegation is a technique in Javascript where a single event listener is placed on a common ancestor(usually a parent) of multiple elements, and it leverages event bubbling to handle events for all the child elements.
