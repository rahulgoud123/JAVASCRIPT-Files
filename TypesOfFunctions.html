Arrow Function

Arrow function is a concise way of writing the function and it is introduced in ecma script

An anonymous function in Javascript is a function without name. It is often used in situations where a function is needed temporarily, such as passing a function as an agument to another function.

In javascript, variables can have either global or local scope. Global scope means the variable is accessible throughout the entire program, while local scope means the variable is only accessible within a specific block or function.

A closure is the coiation of function bundled together (enclosed) with references to its surrounding state (the lexical environment). In other words, a closure gives you access to an outer function's scope from an inner function.

In javascript, a pure function is a function that, givn the same input, will always return the same output and has no side effects.(does not modify the state)
